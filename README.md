# Multi LED Indicator Board

**Description:**  
Industrial LED Indicator Panel Simulation project for CAPE Electric type company.

**Components:**  
- 5 LEDs (POWER, READY, RUN, FAULT, E-STOP)  
- 5 Resistors (220Î©)  
- Power supply (5V or 9V)  
- Breadboard / PCB layout (Single layer)

**Files Included:**  
- AutoCAD drawings (FrontPanel.dwg, CircuitDiagram.dwg)  
- Images of layout (optional)  
- Optional Arduino code for LED sequence

**Notes:**  
- AutoCAD files are design only; to run the project, connect components on Breadboard or PCB.
